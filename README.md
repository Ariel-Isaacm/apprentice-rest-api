## Nearsoft Apprentice API
Sample Nodejs/Express rest api

## How to run it
* In your terminal run `npm install`
* Once npm packages are installed run `npm start`

##Endpoints
Method | Routes
------------ | -------------
GET | /
GET | /games
POST | /game
GET | /game/:id
PUT | /game/:id
DELETE | /game/:id
GET | /games/ps4
GET | /games/cheap
